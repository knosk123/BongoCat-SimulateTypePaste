# BongoCat 桌宠粘贴模式

这是一个 Windows 桌宠粘贴工具，基于以下两个项目/思路结合开发：

1. BongoCat 桌宠项目：[ayangweb/BongoCat](https://github.com/ayangweb/BongoCat)
2. SimulateTypePaste 程序思路：模拟键盘输入剪贴板文本

本项目保留 BongoCat 原有桌宠功能，并在其基础上额外增加“桌宠粘贴模式”。开启后，用户可以右键桌宠，将当前剪贴板中的文本以模拟键盘输入的方式输入到当前已聚焦的应用中。

## 主要功能

- 保留 BongoCat 原有桌宠、托盘、设置、窗口置顶、窗口穿透等功能
- 新增“桌宠粘贴模式”开关
- 开启粘贴模式后，右键桌宠可模拟输入剪贴板文本
- 支持中文文本输入
- 支持窗口穿透/忽略鼠标事件，但窗口穿透开启时鼠标事件不会传给桌宠，因此右键粘贴不会触发

## 使用方法

1. 下载 Release 中的 Windows 安装包
2. 安装并启动 BongoCat
3. 打开设置或托盘菜单，开启“桌宠粘贴模式”
4. 先点击你想输入文字的目标窗口，让目标窗口获得焦点
5. 右键桌宠，程序会把剪贴板文本模拟输入到目标窗口

## 注意事项

- 如果开启了“窗口穿透/忽略鼠标事件”，鼠标点击会穿过桌宠，右键粘贴不会触发
- 如需移动桌宠位置，请先关闭窗口穿透，移动后再开启
- 本仓库 Release 只发布安装包和说明，不包含本地开发缓存、日志、个人配置或隐私文件
